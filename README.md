# 📱 Quran App

A Flutter-based Islamic application that provides users with a complete Quran experience.

---

## 🎥 Demo

<img width="480" height="270" alt="Quran Git" src="https://github.com/user-attachments/assets/78483781-60f3-437f-8c9a-1ec9ebaa07b6" />


## 🚀 Features

- 📖 Full Quran with Surah details  
- 🕌 Hadith section  
- 📻 Quran Radio streaming  
- 🧭 Prayer Times  
- 📿 Digital Sebha  
- 🕓 Recently read Surahs  

---

## 🛠️ Tech Stack

- Flutter  
- Dart  
- Shared Preferences
